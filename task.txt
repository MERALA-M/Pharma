public interface MedicineComponent { void displayHierarchy(String indent); }
public class MedicineCategory implements MedicineComponent {
    private List<MedicineComponent> children = new ArrayList<>();
    public void add(MedicineComponent c) { children.add(c); }
    public void displayHierarchy(String indent) {
        for (MedicineComponent c : children) c.displayHierarchy(indent + " ");
    }
}
TRIGGER:

MedicineController.handleShowComposite ("Hierarchy" button).
UML RELATIONSHIPS:
Medicine and MedicineCategory implement MedicineComponent.
MedicineCategory contains list of MedicineComponent (Aggregation).

public interface InventoryObserver { void onStockAlert(Medicine m, String msg); }
public class InventorySubject {
    private List<InventoryObserver> observers = new ArrayList<>();
    public void notifyObservers(Medicine m) {
        for (InventoryObserver o : observers) o.onStockAlert(m, "Low Stock");
    }
}
TRIGGER:

DashboardController registers as observer to show live alerts.
UML RELATIONSHIPS:
InventorySubject stores InventoryObserver (Aggregation).
DashboardController implements InventoryObserver.
